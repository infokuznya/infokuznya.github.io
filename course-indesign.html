Курс “Adobe InDesign” базовый

(обзорные темы в объеме нескольких занятий)

#1. Введение в компьютерную графику. Знакомство с программой

  • задачи курса
  • первое знакомство с программой
  • создание нового документа
  • окно документа, перемещение по нему
  • основные инструменты

#2. Рабочая область программы и палитра инструментов

  • обзор интерфейса
  • палитры
  • обзор меню программы
  • установка единиц измерения, линейки
  • применение направляющих
  • панель инструментов
  • настройка и сохранение рабочей области
  • параметры просмотра
  • контекстное меню и "горячие" кнопки

#3. Работа с текстом

  • типы шрифтов
  • гарнитура, кегль, интеринльяж
  • основные приемы форматирования текста

Практическое задание #1: макет листовки

#4. Манипуляции с бъектами

  • расположение на странице
  • слои
  • выравнивание и равномерное распределение
  • работа с координатами
  • трансформации (масштаб, вращение, деформация, дублирование, зеркальное отображение)

Практическое задание #2: верстка буклета (часть 1 - макет)

#5. Работа с графикой

  • типы файлов, поддерживаемые InDesign
  • размещение графики в публикации
  • обтекание текстом
  • модульная сетка

Практическое задание #2: верстка буклета (часть 2 - добавляем графику)

#6. Работа с цветом

  • общее о цветовых моделях
  • возможности работы с цветом
  • подробнее о СМУК
  • редактирование цвета объектов с помощью палитры образцов
  • знакомство с палитрой цветов ©PANTONE
  • применение готовых библиотек красок

#7. Создание многостраничного документа

  • грамотная разметка документа
  • манипуляции в панели страниц оттдельными страницами документа (вставка, удаление, перемещение, нумерация, добавление разделов)
  • первый спецсимвол “автонумерация”
  • мастер-страницы и варианты применения

Практическое задание #3: макет брошюры

#8. Работа с таблицами

  • создание
  • редактирование
  • оформление

Практическое задание #4: макет документа с таблицами (прайс услуг)

#9. Работа со стилями

  • создание стиля абзаца
  • создание стиля для символа
  • разметка текста стилями
  • изменение оформления публикации с помощью стилей

#10. Вывод документа в PDF

  • обзорно о допечатной подготовке
  • предварительная проверка публикации
  • сборка для переноса в типографию
  • окно "Печать"
  • создание корректного файла PDF, для передачи в репроцентр
(проверка файлов PDF, рассматривается в полном курсе по допечатной подготовке)

Практическое задание: сборка публикации и вывод файла PDF

Итоговоя самостоятельная работа

Верстка макета журнала. (1-2 занятия)

По окончании курса слушатель будет уметь:

Использовать инструментарий программы

Создавать макеты различной сложности
используя текстовые стили

Совмещать векторные и растровые элементы в своих работах

Создавать привлекательные макеты, как одностраничные так и много страничные
(например: визитки, буклеты, журналы, книги и т.д)
